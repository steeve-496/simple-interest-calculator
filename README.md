## How to Use

1.  Open the live demo link or the `index.html` file in your browser.
2.  Enter the **Principal** amount in the first field.
3.  Enter the **Annual Interest Rate** (in %).
4.  Enter the **Time** period (in years).
5.  Click the **"Calculate"** button.
6.  The **Simple Interest** and **Total Amount** will be displayed in the results area.
7.  Click **"Reset"** to clear all fields for a new calculation.
